Reading Notes: What is cell type and how to define it

Created2024-01-14|Updated2024-01-14|RNA-seq Reading Notes
|Word Count:840|Reading Time:3mins|Post Views:

PREFACE

Why read this review?

  • 在进行细胞类型注释的时候试图仅根据marker来推断一种细胞类型，因而发现了对细胞分类理解的不足。
  • 如何对细胞类型进行注释：看别人选择的标记基因，根据知识储备选择标记基因

What is the biological significances of this review?

  • 在于比较不同细胞类型在功能上的不同，了解健康和患病条件下细胞功能的动态变化
  • 因此不能单单根据marker来推断一种细胞类型。

General view of this review：

  • 这篇是cell的一篇综述，讲述什么是细胞类型以及如何定义细胞类型。
  • 细胞类型是一个比较复杂的概念，和进化、表观遗传、转录表达、细胞状态都有很强的联系。

SUMMARY

Cell types are the basic functional units of an organism. Cell types exhibit diverse phenotypic properties at multiple levels, making them challenging to define, categorize, and understand. This review provides an overview of the basic principles of cell types rooted in evolution and development and discusses approaches to characterize and classify cell types and investigate how they contribute to the organism’s function, using the mammalian brain as a primary example. I propose a roadmap toward a conceptual framework and knowledge base of cell types that will enable a deeper understanding of the dynamic changes of cellular function under healthy and diseased conditions.

细胞类型是有机体的基本单元。细胞类型在多个层面上表现出不同的表型特性，这使得它们的定义、分类和理解具有挑战性。这篇综述概述了植根于进化和发育的细胞类型的基本原理，并以哺乳动物大脑为主要例子，讨论了表征和分类细胞类型的方法，以及研究它们如何对生物体的功能做出贡献。我提出了一个通往细胞类型概念框架和知识库的路线图，这将使我们能够更深入地了解健康和患病条件下细胞功能的动态变化。

INTRODUCTION

  • 了解生物体内细胞的结构和功能对于理解其整体功能至关重要。
  • 将细胞分类将细胞分类大大降低了研究细胞结构和功能的复杂性。
  • 缺乏一致和标准的细胞类型定义对于可重复的研究提出了挑战。
  • 最近单细胞转录组学的进展改变了细胞分类方式，但在分类的准确性上仍存在问题。
  • 该综述主要以哺乳动物大脑为例，探讨了有关细胞类型定义的关键问题。

行文思路：

  1. 细胞结构功能对生物体功能很重要
  2. 细胞分类可以研究细胞结构功能
  3. 细胞分类标准存在问题；单细胞转录组改变了分类方式，但是也有问题
  4. 因此本文介绍和解决细胞分类定义的问题。

APPROACHES TO CHARACTERIZE CELL TYPES

–表征细胞类型的方法

Cell types in the brain and the body exhibit diverse properties in many modalities – molecular, morphological, physiological, and functional.

大脑和躯体中的各种细胞类型在分子的（molecular）、形态的（morphological）、生理的（physiological）和功能的（functional）许多形式方面（modalities）都表现出不同的特性。
